╔════════════════════════════════════════════════════════════════════════════╗
║                           FLUJO DE EJECUCIÓN                              ║
╚════════════════════════════════════════════════════════════════════════════╝

1️⃣  USUARIO
    ↓ (Escribe consulta)
    
2️⃣  AGENTE (backend/agent.py)
    ├─ Valida datos sensibles (Guardrail - IE6)
    ├─ Procesa consulta con LLM
    ├─ Selecciona herramienta apropiada
    │  └─ CALLBACK MONITOREO INICIA (IE1-IE4)
    │     ├─ Registra timestamp
    │     ├─ Mide latencia
    │     ├─ Captura RAM usada
    │     └─ Detecta errores
    ├─ Ejecuta herramienta
    │  └─ CALLBACK MONITOREO FINALIZA
    │     └─ Escribe logs/agent_logs.jsonl
    └─ Retorna respuesta a usuario
    
3️⃣  MEMORIA (MemoriaCompuesta)
    ├─ Corto Plazo: Almacena conversación actual
    └─ Largo Plazo: Agrega resumen a FAISS (vector_db/)
    
4️⃣  DASHBOARD (dashboard.py)
    ├─ Lee logs/agent_logs.jsonl
    ├─ Genera gráficos Plotly
    └─ Muestra en Streamlit (http://localhost:8501)
