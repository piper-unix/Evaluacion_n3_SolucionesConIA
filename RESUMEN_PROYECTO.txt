🚀 INICIO RÁPIDO (3 OPCIONES)
═══════════════════════════════════════════════════════════════════════════════

OPCIÓN 1: WINDOWS
  1. Edita .env con tu OPENAI_API_KEY
  2. setup.bat install
  3. setup.bat agent (Terminal 1)
  4. setup.bat dashboard (Terminal 2)
  5. Abre http://localhost:8501

OPCIÓN 2: LINUX/MAC
  1. Edita .env con tu OPENAI_API_KEY
  2. ./setup.sh install
  3. ./setup.sh agent (Terminal 1)
  4. ./setup.sh dashboard (Terminal 2)
  5. Abre http://localhost:8501

OPCIÓN 3: DOCKER
  1. Edita .env con tu OPENAI_API_KEY
  2. docker-compose up --build
  3. Abre http://localhost:8501
